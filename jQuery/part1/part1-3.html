Traversing elements

Da biste izmenili HTML elemente koje ste selektovali morate proći (traversirati) kroz njih.

Lista svih jQuery traversing metoda možete naći na http://api.jquery.com/category/traversing/

Najčeše korišnjene traversing metode su:

  • each()
  • first()
  • last()
  • next()
  • prev()
  • parent()
  • children()
  • siblings()

Na primer imamo sledeću HTML strukturu 

<div class="contentDiv">Div</div>
<div class="contentDiv">Div</div>
<div class="contentDiv">Div</div>

Sad želimo da ih izaberemo selektorom $('div.contentDiv'), da prođemo kroz sve elemente i da nešto radimo sa elementima.

each()

each() metoda vam dozvoljava da prođete kroz sve elemente koje ste selektovali. 

$(".contentDiv").each(function(index,element){
    $(element).css('background-color: #ff9999');
});

U prethodom primeru selektujemo sve elemente sa klasom contentDiv i prolazimo kroz njih uz pomoć each metode i svakom od selektovainh elementata menjamo boju pozadine.

U each() metodi definišemo callback funkciju koja ima parametre index i element. Index je index elementa u odnosu na njegov parent, a ne cele HTML strukture. Drugi paretam funkcije, element, je DOM tipa i zato se mora staviti u $() funkciju da bismo napravili jQuery objekat.

callback funkcija ne mora imati parametre index i element. U slučaju da element parametar nije prosleđen možemo koristiti this. this se odnosi na DOM objekat u kome se trenutno nalazimo.

U sledećem primeru prolazimo kroz sve .contentDiv elemente i menjamo sadržaj svakog dita tako što na već postojeći sadržaj dodajemo index elementa 

$(".contentDiv").each(function(index){
    $(this).text($(this).text() + " " + index);
});

first()

first() metoda vraća prvi element u nizu. 

$(".contentDiv").first(function(index,element){
    $(element).css('background-color: #ff9999');
});
//setuje pozadinu prvom elementu sa klasom .contentDiv

last()

first() metoda vraća poslednji element u nizu. 

$(".contentDiv").last(function(index,element){
    $(element).css('background-color: #ff9999');
});
//setuje pozadinu poslenjem elementu sa klasom .contentDiv

next()

Selektuje sledeći HTML element u DOM stablu. 

var firstDiv = $(".contentDiv").first();
firstDiv.next().css('background-color: #ff9999');

prev()

Selektuje prethodni HTML element u DOM stablu. 

var firstDiv = $(".contentDiv").last();
firstDiv.prev().css('background-color: #ff9999');

parent()

Selektuje parent element objekta nad kojim je parent pozvan. 

$("#theDiv").parent();

children()

Selektuje sve child elemente. 

var children = $("#theDiv").children();
children.each(function(){
    //TODO
});
//selectuje sve child elemente elementa koji ima id="theDiv" i prolazi kroz njih

siblings()

Ova funkcija vraća sve elemente unutar parent-a elementa nad kojim je siblings finkcija pozvana isključujući taj element. 

var siblings = $("#theDiv").siblings();
siblings.each(function(){
    //TODO
});

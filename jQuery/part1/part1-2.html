Selecting elements

jQuery ima širok spektar mogućnosti selektovanja elemenata, a kad ih odaberete možete iterirati kroz njih i pozivati razne funkcije.

Lista svih jQuery selectora može se naći na http://api.jquery.com/category/selectors/

jQuery vam omogućuje odabir elemenata na osnovu sledećih kriterijuma.

  • Element name
  • Element id
  • Element CSS class
  • Element attributes
  • Element visibility
  • Element order
  • Form fields
  • Element parents and children
  • Combinations of above

Selectovanje elemenata uz pomoć jQuery-ja se radi preko funkcija $ ili jQuery. String koji prosleđujemo funkcijama se naziva selector. 

$('div'); //div je selector

Element name

Element name je ustvari naziv HTML taga. Tako da ako kao selector prosledite a, jQueary će vam selectovati sve a tagove. 

$('a'); //pravi objekte svih a tagova

Element id

Ako vam treba neki specifičan element onda taj element možete selectovati po njegovom id-u. Taraba (#) se stavlja ispred id stringa. 

$('#anchorID'); //selektuje <a id="anchorID"></a>

Element CSS class

jQuery vam omogućuje selektovanej elemenata sa odrećenom klasom. Ispred naziva klase stavljamo tačku (.). 

$('.anchorClass'); //selektuje <a class="anchorClass link"></a>

Element Attributes

Da biste selektovali elemente sa određenim atributom onda naziv atributa stavljate u [] zagradama 

$('[height]'); // Selektuje sve elemente koji imaju definisan atribut height
$('[height="200"]'); /* Selektuje sve elemente koji imaju definisan atribut 
                        height i gde je height ima vrednost 200 
                        Vrednost treba stavljati pod navodnicima u slucaju da 
                        se vrednost sastoji iz više od jedne reči.*/

Element visibility

Za selectovanje na osnovu njihovih "vidljivosti" postoje dva selectora. :visible i :hidden 

$(':visible'); // Selektuje sve vidljive elemente
$(':hidden'); // Selektuje sve sakrivene elemente

Element order

jQuery vam omogućuje selektovanje elemenata na osnovu njihovog redosleda. :first i :last selectuju prvi i poslednji element. :event i :odd selektuju paran i neparan element. :eq(x) selektuje element sa indexom x (prvi je 0). :gt(x) selektuje sve elemente sa indexom većim od x, dok :lt(x) selektuje sve elemente sa idexom manjim od x. 

$('div:even'); // Svi parni div-ovi

Form Fields

jQuery ima ugrađenu podršku za selektovanje elemenata forme. :input selektor selektuje sva input polja (text, radio, checkbox, select, textarea, button). :text selektuje sve iput-e tipa text ali ne selktuje textarea polja. :radio selektuje sva radio polja. :checkbox selktuje sva checkbox polja. :checked selektuje sva checkbox ili radio polja koja se checkirana. 

$(':text'); // svi input tagovi gde je type="text"

Element parents and children

jQuery vam omogućuje selekciju elemenata na osnovu njihove parent-child relacije.

:first-child i :last-child selektuji prvi, tj. poslednji element njihovog parenta. 

$('div:last-child'); // Poslenji div element svog parenta

parent > child selector vam omogućuje selectovanje child elemenata određenog parent-a 

$('div > p'); // selectuje sve p elemente koji se nalaze u div elementu

Combinations of above

Sve ove selektore odozgo možemo kombinovati i od njih napraviti složenije selektore. 

$('div[attrib="some div"] > p:eq(6) > a#anchorID[href="#"]'); 
/* selectuje sve a tagove sa id-em anchorID i href-om # koji se nalaze u p tagu
   koji je sedmi (za eq(0) kažemo da je prvi) child div parent-a koji ima atribut 
   attrib i gde je vrednost tog atributa some div. */

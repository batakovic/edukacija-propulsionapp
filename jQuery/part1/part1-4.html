CSS manipulation

Uz pomoć jQuery-ja možete manipulisati CSS atributama i klasama selektovanih HTML elemenata.

Kompletna lista funkcija za CSS manipulaciju se može naći na http://api.jquery.com/category/css/

Reading CSS attributes

Sledeći primer pokazuje kako možemo izvući boju pozadine nekog elementa 

var bcgColor = $('div#theDiv').css('background-color');

Changing CSS attributes

U sledećem primeru vidimo kako se setuje boja pozadine nekog elementa 

$('div#theDiv').css('background-color', '#ff0011');

Drugi parametar css() funkcije je vrednos na koju setujem css atribut (prvi parametar).

Jednim pozivom css() funkcije možemo setovati više atributa koristeći JavaScript objekat 

$('div#theDiv').css(
    {
        'background-color' : '#ff0011',
        'border' : '1px sold #0044ff'
    }
);

CSS classes

Uz pomoće jQuery-ja moguće je menja ti klase html elemenata. 

$('div#theDiv').addClass('newClass'); //dodavanje klase
$('div#theDiv').addClass('newClass1 newClass2'); //dodavanje više klasa

Moguće je i "togglovati" klasu. To znači dodati je ako je nema i izbrisati je ako je ima. 

$('div#theDiv').toggleClass('newClass');
$('div#theDiv').toggleClass('newClass1 newClass2');

hasClass() metoda proverava da li element ima zadatu klasu 

if($('div#theDiv').hasClass('newClass')){
    //TODO
};
